Button is the brand's single CTA primitive — pill-shaped, four variants.

```jsx
<Button variant="primary">Call Now</Button>
<Button variant="secondary">View Services</Button>
<Button variant="outline">Learn More</Button>
<Button variant="ghost" as="a" href="tel:5599627503">(559) 962-7503</Button>
```

Reserve `variant="primary"` (growth red) for the single most important action on a screen — booking, calling, or starting a return. Use `secondary`/`outline` for everything else.
