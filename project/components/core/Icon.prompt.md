Icon renders a single-color Lucide glyph by name — use for contact methods, service categories, and inline UI affordances.

```jsx
<Icon name="phone" size={20} color="var(--icon-primary)" />
```

Requires the consuming page to load the Lucide CDN script before mount:
`<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.js"></script>`

Common names used across this design system: `phone`, `mail`, `globe`, `map-pin`, `clock`, `check`, `check-circle-2`, `chevron-down`, `chevron-right`, `file-text`, `briefcase`, `home`, `building-2`, `calculator`, `stamp`, `fingerprint`, `scale`, `shield-check`.
