ServiceCategoryCard groups one service line (Income Tax / Notary / Livescan) with an icon header and checklist of sub-services.

```jsx
<ServiceCategoryCard
  icon="calculator"
  title="Income Tax"
  items={['Individual', 'Small Business / Self Employed', 'ITIN Applications', 'Audit Services']}
  note="Virtual/online tax preparation available any day, all year."
/>
```

Lay 3 of these in a grid for the Services section — one per C&R service line.
